# PWM LED Control (AVR)

This project demonstrates PWM (Pulse Width Modulation) using an AVR microcontroller to control LED brightness.

## Features
- Fast PWM using Timer0
- Duty cycle control
- LED brightness variation

## Hardware
- ATmega microcontroller
- LED + resistor

## Description
PWM signal is generated using Timer0 in Fast PWM mode. The duty cycle controls LED brightness.
